## **简介**

本项目使用pyqt5为yolov5-6.0开发了一个GUI界面，共实现了图片检测、视频检测、摄像头检测、选择权重四个功能，并将图片检测与视频检测合并为了同一个按钮。刚接触YOLO不久，代码很多地方写的不规范，仅供参考。

## **代码执行**

```
pip install -r requirements.txt
python window.py
```
